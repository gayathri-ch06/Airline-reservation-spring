# Airline-reservation-spring
A web platform based on SpringMVC and MySql to book one-way or the roundtrips from a source to destination. 

# Tech Stack Used
Front-end : HTML, CSS, JS
Backend : Spring MVC
Database : MySQL

# How to run the code
1) Install STS extension for either eclipse or VScode. Both the extensions are available on https://spring.io/tools
2) Import the project as spring starter project.
3) Run the project as a spring applications and access the site at http://localhost:3200
